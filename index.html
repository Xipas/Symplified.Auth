About Symplified.auth

Symplified.auth provides enterprise identity and authentication capabilities to developers using the Xamarin Studio IDE for cross-platform mobile development. Using the Symplified.auth library, users can sign into the native mobile app using existing credentials. Users may authenticate via third-party identity providers (IdP) using the SAML 2.0 and OAuth 1.0/2.0 standards.

Symplified.auth is written in C#, the native language used in Xamarin Studio. Sample projects for Android and iOS are provided.
